Arg-1 : Required. The name of the tag.
Arg-2 : The name of the release.
Arg-3 : Text describing the contents of the tag. Basically, the description of the release.
Arg-4 : OAuth token generated.
	( Create one at https://github.com/settings/ ) 
Arg-5 : Owner of the repo.
Arg-6 : Name of the Repo.

USAGE :

sh git-release.sh Arg-1 Arg-2 Arg-3 Arg-4 Arg-5 Arg-6
